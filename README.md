# Smart Banking System (Core Java)

## Overview

A console-based banking system built using Core Java without using collections (ArrayList/HashMap), focusing on fundamental programming concepts.

## Key Highlights

* Implemented using arrays instead of collections
* Manual data handling and memory management
* Clean modular structure (Model-Service architecture)

## Features

* Create Account
* Deposit & Withdraw Money
* Transaction History Tracking
* Exception Handling for invalid operations

## Concepts Used

* Object-Oriented Programming (OOP)
* Arrays and Manual Data Structures
* Exception Handling
* Layered Architecture (Basic)

## Tech Stack

* Java (Core)


## How to Run

1. Clone the repository
2. Open in IDE (IntelliJ / Eclipse/ VS-code)
3. Run Main.java
